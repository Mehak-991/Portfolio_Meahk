### 👋 About Me
I'm an **Aspiring AI/ML Engineer & Data Science Enthusiast**. I specialize in building intelligent applications using **NLP**, **Machine Learning**, and **Generative AI**. My portfolio showcases my ability to bridge the gap between complex data models and high-performance user interfaces.

### 🚀 Key Projects

#### [1. CineMatch AI — Movie Recommendation System](https://github.com/Mehak-991/Movie-Recommendation-System-using-NLP-TF-IDF)
*Content-based recommendation pipeline using NLP.*
- **Tech:** Python, Django, Scikit-Learn, TF-IDF, SVD.
- **Highlights:** Sub-50ms response times for 1M+ records, fuzzy-matching search.

#### [2. California House Price Predictor](https://github.com/Mehak-991/California-House-Price-Prediction-Model)
*ML model for real estate price forecasting.*
- **Tech:** Python, Scikit-Learn, Random Forest, GridSearchCV.
- **Highlights:** 5-fold cross-validation, test RMSE of ~$47K.

#### [3. SMS Spam Guard](https://github.com/Mehak-991/SMS-Spam-Classification-System-using-NLP)
*Real-time SMS classification system.*
- **Tech:** NLTK, Naive Bayes, Streamlit.
- **Highlights:** 97.10% accuracy, 100% precision on test data.

#### [4. SEO Engine AI](https://github.com/Mehak-991/SEO-Engine-AI)
*AI-driven content automation system.*
- **Tech:** FastAPI, Gemini 2.0 Flash, Playwright, React.
- **Highlights:** Intent-aware blog generation, stealth scraping of e-commerce trends.

### 🛠️ Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/Mehak-991/portfolio.git
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Run development server**
   ```bash
   npm run dev
   ```
